⚙️ 2. Kết nối máy mới

Thực hiện trên máy nhân viên cần quản lý:

1

Tải file cài đặt

Truy cập yidinginternational.com → đăng nhập → vào tab IT Agent → nhấn nút 📥 Tải xuống Agent. File install.bat sẽ được tải về.

2

Chạy với quyền Administrator

Chuột phải vào install.bat → Run as Administrator. Script sẽ tự động cài agent và thiết lập khởi động cùng Windows.

⚠️ Bắt buộc phải chạy với quyền Administrator. Nếu không, agent sẽ không tự khởi động khi bật máy.
3

Xác nhận kết nối

Sau khi cài xong, máy sẽ tự kết nối VPS. Vào Dashboard IT Agent tại yidinginternational.com — tên máy sẽ xuất hiện trong danh sách bên phải với Connected.

Cơ chế khởi động: Agent được ghi vào Registry HKLM\Software\Microsoft\Windows\CurrentVersion\Run. Mỗi lần Windows khởi động, agent tự chạy ngầm và kết nối VPS. Nếu mất mạng, agent tự thử lại mỗi 10 giây.

🛠️ 4. Các chức năng điều khiển

Chọn máy cần điều khiển từ danh sách bên phải, sau đó nhấn các nút tương ứng:

📷
Screenshot (Chụp màn hình)
Chụp toàn bộ màn hình máy nhân viên ngay lập tức. Kết quả hiển thị là ảnh, click để mở full size.
📸
Camera (Chụp webcam)
Kích hoạt webcam máy nhân viên và chụp 1 ảnh. Cần máy có webcam và không bị phần mềm khác chiếm dụng.
📊
System Info (Thông tin hệ thống)
Xem CPU%, RAM%, Disk% đang dùng. Kiểm tra nhanh tình trạng tải của máy.
📋
Processes (Tiến trình)
Danh sách 50 tiến trình đang chạy trên máy, sắp xếp theo RAM sử dụng. Xem ai đang chạy gì.
🌐
Network Info (Thông tin mạng)
Hostname và địa chỉ IP nội bộ của máy nhân viên. Dùng để xác định vị trí máy trong mạng.
🔔
Send Notification (Gửi thông báo)
Gửi popup thông báo xuất hiện trên màn hình nhân viên. Nhập nội dung → Enter hoặc nhấn nút Gửi.
⚡
PowerShell
Chạy bất kỳ lệnh PowerShell nào trên máy nhân viên. Kết quả trả về dạng text. Mặc định timeout 30 giây.

Nút "Tất cả máy"

Bên cạnh System Info và Notification có nút 📢 Tất cả máy — gửi lệnh đến tất cả máy đang online cùng lúc. Thường dùng cho thông báo toàn công ty hoặc kiểm tra hàng loạt.

🖨️ 5. Quản lý máy in

📄
Liệt kê máy in
Hiển thị tất cả printer đã cài trên máy nhân viên, kèm trạng thái và cổng kết nối.
📋
Hàng đợi in
Xem danh sách tài liệu đang chờ in. Xem ai đang in gì, trạng thái job.
🗑️
Xoá hàng đợi
Xoá tất cả job in đang pending. Dùng khi máy in bị kẹt hoặc có job lỗi.
🖨️
Mở rộng panel quản lý
Nhấn nút 🖨 để mở/đóng sub-panel quản lý printer với đầy đủ các chức năng bên dưới.
➕
Cài máy in qua IP
Thêm network printer vào máy nhân viên từ xa. Nhập IP máy in + Tên hiển thị → Cài đặt.
📤
In file
In một file trên máy nhân viên ra printer. Nhập đường dẫn file (trên máy nhân viên) + tên printer.
Ví dụ cài máy in: IP = 192.168.1.50, Tên = Máy in tầng 2 → nhấn Cài đặt. Printer sẽ xuất hiện trong danh sách của máy nhân viên.

👤 6. Avatar & Tên IT Admin

Thay đổi tên hiển thị

Trong panel phải, nhấn icon ✏️ cạnh tên admin → nhập tên mới → nhấn ✓ để lưu. Tên này sẽ xuất hiện làm tiêu đề thông báo khi gửi notification.

Thay đổi avatar

Nhấn vào vòng tròn avatar → chọn file ảnh. Avatar được lưu trong browser và tự đồng bộ sang tất cả máy agent đang online.

⚠️ Lưu ý về avatar: Avatar lưu trong localStorage của browser. Nếu dùng máy tính khác hoặc xóa dữ liệu browser, cần upload lại. Avatar sau khi upload sẽ được đẩy sang các máy agent để hiển thị nhất quán.

❓ 7. Xử lý sự cố

Triệu chứngNguyên nhânCách xử lý
Máy không hiện trong danh sách Agent chưa chạy hoặc máy tắt Bật máy nhân viên, đảm bảo có internet. Hoặc chạy lại install.bat.
Trạng thái "Đang kết nối..." không đổi VPS relay bị lỗi hoặc web chưa load xong Chờ 10-15 giây, refresh trang, mở lại tab IT Agent.
Lệnh gửi nhưng không có kết quả Agent bị crash hoặc mất kết nối Kiểm tra máy nhân viên còn online không. Thử gửi lại sau 10 giây.
Camera trả về lỗi Máy không có webcam hoặc bị ứng dụng khác chiếm Đóng Zoom/Teams/Meet trên máy nhân viên trước khi thử.
PowerShell không trả về gì Lệnh chạy quá 30 giây (timeout) Dùng lệnh ngắn hơn hoặc thêm -timeout 60 trong params.
Máy in không tìm thấy Driver chưa cài hoặc printer offline Kiểm tra máy in bật và kết nối mạng. Thử dùng Get-Printer qua PowerShell.

Kiểm tra agent có đang chạy không

Mở Task Manager trên máy nhân viên → tab Details → tìm tiến trình pythonw.exe. Nếu không thấy, agent đã tắt. Chạy lại bằng cách: mở C:\YiDingAgent\agent.py bằng pythonw.exe hoặc khởi động lại Windows.
